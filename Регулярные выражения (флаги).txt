r+ — открыть файл для чтения и для записи.

w+ — открыть файл для чтения и для записи, установив указатель потока в начало файла. 
Если файл не существует — он создаётся.

a — открыть файл для записи, установив указатель потока в конец файла. 
Если файл не существует — он создаётся.

a+ — открыть файл для чтения и записи, установив указатель потока в конец файла. 
Если файл не существует — он создаётся.
